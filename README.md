# English Cards (Mobile App)

Приложение для заучивания английских слов с использованием карточек. Проект ориентирован на мобильные устройства и кроссплатформенную работу.

## Стек технологий
* **Core:** Vue 3
* **UI Components:** Ionic Framework
* **Hybrid Engine:** Capacitor (Android поддержка)
* **Build Tool:** Vite

## Основной функционал
* Интерактивные карточки: просмотр слова и перевода по клику.
* Навигация по списку слов.
* Адаптивный мобильный интерфейс.
